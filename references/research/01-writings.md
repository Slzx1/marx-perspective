# 卡尔·马克思著作调研：以《资本论》为核心

> 调研日期：2026-05-26
> 框架：《资本论》为主线，其余著作作为补充和背景
> 可靠性标注：一手=马克思原著 | 二手=学术研究 | 推测=基于一手/二手推理但不确证

## 一、《资本论》体系总览

### 1.1 三卷总体结构

《资本论》的副标题是"政治经济学批判"，三卷分别对应资本运动的不同阶段：

| 卷 | 标题 | 发表年份 | 核心对象 | 逻辑位置 |
|---|---|---|---|---|
| 第一卷 | 资本的生产过程 | 1867（马克思生前出版） | 直接生产过程 | 本质/存在层面：价值、剩余价值在直接性中被揭示 |
| 第二卷 | 资本的流通过程 | 1885（恩格斯编） | 流通过程 | 经过中介的直接性：资本循环与周转 |
| 第三卷 | 资本主义生产的总过程 | 1894（恩格斯编） | 生产与流通的统一 | 现实/表面层面：利润、利息、地租——在经验社会中呈现的具体形式 |

> 逻辑结构源自马克思在《大纲》导言中的方法论："从抽象上升到具体"。三卷不是按历史顺序排列，而是按范畴在发达的资本主义社会中的结构关系排列。**来源：一手，《资本论》原著 + 二手，Ilyenkov《马克思〈资本论〉中抽象与具体的辩证法》**。

### 1.2 写作过程与恩格斯编辑争议（重大）

**一手事实：** 马克思去世时仅出版了第一卷。第二卷有8份手稿（1865-1881年，约1700页），第三卷有一份1864-65年的主体草稿加若干补充碎片。恩格斯在临终嘱托下花了11年编辑出版了第二卷（1885）和第三卷（1894）。

**二手学术争议（基于MEGA原始手稿对比）：**

| 批评者（如Michael Heinrich） | 辩护者（如Vitaly Vygodsky） |
|---|---|
| 恩格斯改动了术语、重组了章节、删除了括号中的试探性表述，使手稿显得比实际更完整 | 恩格斯面对的任务是将"研究"转化为"表述"，这是必要且合理的编辑工作 |
| 马克思关于危机的论述是试探性的，括号中写的；恩格斯删除括号并添加章节标题，创造了貌似完成的"危机理论" | 恩格斯忠实地贯彻了马克思的精神，批评者混淆了研究方法与叙述方法的区别 |
| 在"崩溃"（Zusammenbruch）问题上，恩格斯将马克思的口语化表述"zum Klappen bringen"改成了更确定性的"zum Zusammenbruch bringen" | 恩格斯注明的修改范围已被自己所承认 |
| 第三卷第五篇（信用）被从马克思的"第五点"扩展为11个章节，改变了系统重心 | 考茨基等后来编辑者指出：谁能保证自己的版本比恩格斯更接近马克思的意图？ |

**矛盾记录：** 恩格斯自己存在矛盾表述。他在序言中声称"尽可能逐字逐句"忠于原著、所有重大改动都标记了"F.E."，但同时又承认做了"大量的插入"以使论证清晰、帮助读者理解。这两个目标——保存不完整的草稿 vs. 制作清晰、政治上可用的文本——本质冲突。**来源：二手，MEGA II/4.2原始手稿研究 + Heinrich (1996), Vygodsky (1995), Regina Roth (MEGA编辑)**。

> **影响评估：** 当代严肃的马克思经济学研究日益直接使用MEGA手稿而非仅依赖恩格斯版本。这在危机理论、利润率下降和信用理论领域影响最大。但学者们也普遍认为，没有发现"马克思作为作者和恩格斯作为编辑之间的根本分歧"（Regina Roth语）。**可信度：二手（学术研究），高度可靠。**

## 二、《资本论》第一卷：资本的生产过程

### 2.1 商品与价值理论（第1章）

**核心论点（一手，马克思原著）：**

1. **商品二重性：** 商品是使用价值（满足需求的质）和交换价值（量的交换比例）的统一体。在交换价值背后存在**价值**——由社会必要抽象劳动时间决定。

2. **劳动二重性：** 马克思自称这是"理解政治经济学的枢纽点"。具体劳动创造使用价值，抽象劳动形成价值。
   > 马克思在第1章脚注中写道："对任何种类的劳动同样看待，以一种特殊的实在的抽象为前提……这种抽象本来就是商品生产的社会的特征。"——这表明抽象劳动不是纯思维范畴，而是历史上特定的社会过程的产物。

3. **价值形式分析（第1章第3节）：**
   - 简单/偶然形式：x量商品A = y量商品B（相对价值形式 vs. 等价形式的对立由此出现）
   - 总和/扩大的形式：一种商品的价值用无数其他商品表示
   - 一般形式：所有商品用同一种商品表示其价值
   - 货币形式：一般等价物被社会固定——金，进而国家发行的货币
   > 马克思强调这是**逻辑**展开，不是历史展开。货币是价值的必然表现形式的最终结果。

4. **商品拜物教（第1章第4节）：**
   - **定义：** 在商品生产条件下，人们之间的社会关系采取"物与物之间的关系的虚幻形式"。
   - **不是主观幻觉：** 这是客观的、结构性的现实——因为商品生产者只通过市场交换间接发生关系，社会关系必然被物中介。
   - **来源：一手，《资本论》第一卷第一章。**

**Isaak Rubin的重新发现（二手，学术研究）：**
苏联马克思主义经济学家鲁宾在1923年出版的《马克思价值理论论文集》极大地重塑了对价值形式的理解。其核心洞见：
- 价值不是劳动本身，劳动仅仅是价值的**实体**；价值必须在其与"价值形式"的不可分割的联系中理解。
- 商品拜物教不是第1章的"附录"，而是整个价值形式分析的**高潮**——它展示了在资本主义下，为何社会关系必然采取物的形式。
- **"政治经济学不是物与物关系的科学……也不是人与物关系的科学……而是生产过程中人与人关系的科学。"**

鲁宾的书在斯大林清洗期间被禁（鲁宾被处决），直到1972年被译为英文后，引发了英语世界价值形式理论的复兴。**来源：二手，Rubin (1923/1972)。**

> **价值形式分析与李嘉图的区别：**
> 李嘉图发现了价值背后的劳动，但从未问过*为什么*劳动表现为价值、为什么产品表现为商品。马克思的贡献在于展示形式（交换价值）如何必然从内容（抽象劳动）中产生。这正是"内在批判"——不是从外部标准批判资本主义，而是从其自身范畴的内在矛盾中展开批判。

### 2.2 货币、资本与劳动力的买卖（第2-6章）

**从货币到资本（第4章）：**

马克思区分两种流通：
- 简单商品流通（C-M-C）：卖是为了买，以使用价值为目的
- 资本流通（M-C-M'）：买是为了卖得更贵，以价值增殖为目的

**资本的定义：** "自我增殖的价值"、"过程中的价值、过程中的货币"。

**剩余价值的来源问题：**
- 剩余价值不能来自等价交换（等价交换不产生新价值）
- 也不能来自不等价交换（一方多得即另一方少得，社会总价值不变）
- 因此必然存在一种特殊的商品，其**使用价值本身就是价值的源泉**——这就是**劳动力**

**劳动力与劳动的区别：**
马克思区分了**劳动力**（劳动能力，作为商品出售）和**劳动**（劳动力的实际使用）。这是"政治经济学批判的决定性突破"：
- 劳动力的价值 = 再生产工人所需的生活资料的价值
- 劳动力的使用价值 = 创造超过自身价值的价值的能力
- 差额 = 剩余价值（来源于生产过程而非交换过程）

**来源：一手，《资本论》第一卷第四章。**

### 2.3 剩余价值理论（第7-18章）

**绝对剩余价值 vs. 相对剩余价值：**

| 类型 | 机制 | 手段 |
|---|---|---|
| 绝对剩余价值 | 延长工作日 | 在必要劳动时间固定情况下增加剩余劳动时间 |
| 相对剩余价值 | 缩减必要劳动时间 | 通过提高生产力（机器、技术、协作）降低劳动力价值 |

> 马克思用"工作日"一章生动描述了围绕工作日长度的斗争史、工厂立法等，展示阶级斗争如何实际塑造了资本主义的面貌。

**协作、分工与机器大工业（第13-15章）：**
马克思详细追溯了从手工工场到工厂制度的技术演变——这不是简单的技术史，而是展示机器如何：
1. 将工人变成"机器的附属物"
2. 使资本能够系统地占有"无偿的科学力量"
3. 加剧了"劳动对资本的形式隶属"向"实际隶属"的转化

### 2.4 资本积累与原始积累（第23-25章，第26-33章）

**资本积累的一般规律（第23章）：**
- 资本有机构成（c/v）随积累而提高
- 产生**相对过剩人口**（"产业后备军"）——失业不是因为财富太少，恰是因为资本有机构成提高，对劳动力的相对需求下降
- 一端是财富的积累，另一端是"贫困、劳动折磨、受奴役、无知、粗野和道德堕落的积累"

**原始积累（第26-33章）：**

> 术语：马克思用"所谓原始积累"（Die sogenannte ursprüngliche Akkumulation），以讽刺资产阶级经济学家将其描绘为"勤劳节俭者 vs. 懒惰浪费者"的神话。最新英文译本（Reitter, 2024）采用"original accumulation"而非"primitive accumulation"。

**核心论点（一手，马克思原著）：**
> "资本来到世间，从头到脚，每个毛孔都滴着血和肮脏的东西。"

- 原始积累的本质 = 直接生产者被**暴力**分离于生产资料（特别是土地）
- 方法包括：圈地运动、殖民掠夺、奴隶贸易、国债、税收、保护主义
- **殖民主义和奴隶制被马克思明确称为"原始积累的主要因素"（chief moments）**：
  > "美洲金银产地的发现、土著居民的被剿灭、被奴役和被埋葬于矿井、对东印度开始进行的征服和掠夺、非洲变成商业性地猎获黑人的场所——这一切标志着资本主义生产时代的曙光。"

**来源：一手，《资本论》第一卷第八篇。**

**当代延伸（二手，学术研究）：**
David Harvey发展了"**剥夺性积累**"（accumulation by dispossession）概念，论证原始积累不是一次性历史事件而是持续至今的过程——通过公共服务私有化、土地掠夺、知识产权制度和金融掠夺等新形式。**可信度：二手，高度有影响力但存在争议。**

## 三、《资本论》第二卷：资本的流通过程

### 3.1 资本循环（第1-6章）

马克思分析了三种资本循环形式：
- **货币资本循环：** M-C...P...C'-M'（目的：价值增殖，最清晰表现资本的"生钱"性质）
- **生产资本循环：** P...C'-M'-C...P（目的：再生产，暴露积累来源）
- **商品资本循环：** C'-M'-C...P...C''（目的：满足社会需求，强调生产与消费的联系）

> 三种循环不是分离的，而是同一总过程的三个侧面。只有在三种循环的统一中，总过程的连续性才是可能的。

### 3.2 资本周转与再生产（第7-17章，第18-21章）

- **固定资本 vs. 流动资本：** 不同于不变资本/可变资本——后者基于价值增殖功能，前者基于周转方式
- **再生产图式：** 马克思将社会总生产分为两大部类：
  - 第I部类：生产资料生产
  - 第II部类：消费资料生产
- **简单再生产 vs. 扩大再生产：** 简单再生产的均衡条件：I(v+s) = IIc；扩大再生产要求两大部类不成比例的增长

> **方法论意义：** 第二卷展示了流通带来的可能性——资本循环在**时间中**被中断（卖不出商品、资金断裂）——这为第三卷的危机理论提供了形式上的可能性。

**来源：一手，《资本论》第二卷（恩格斯编）。**

## 四、《资本论》第三卷：资本主义生产的总过程

### 4.1 剩余价值转化为利润、价值转化为生产价格（第1-2篇）

**核心问题——转形问题：**

> 这是马克思经济学中最著名的争议，被称为"转形问题"（transformation problem）。

**矛盾记录（不调和，直接标注）：**

在第一卷中，马克思假定商品按价值交换，利润来源于剥削（s/v）。但现实中，资本有机构成不同的行业会趋向形成**统一的平均利润率**，商品按**生产价格**（而非价值）交换。在第三卷中，马克思尝试将价值"转形"为生产价格。

**Böhm-Bawerk（1896）的经典批判（二手）：**
> 奥地利学派经济学家庞巴维克在《卡尔·马克思及其体系的终结》中指出马克思存在逻辑不一致：(1)如果商品按劳动时间交换，等量资本不可能有等量利润率；(2)如果有等量利润率，商品不可能按劳动时间交换。他称马克思的转形方案为"特设性构造"（ad hoc construction），因为不变资本和可变资本本身也应以生产价格而非价值计算——这是内在的不一致。

**Bortkiewicz（1906-07）的数学批评（二手）：**
> 提出了一个数学上的"修正"，认为马克思的转形不完整——不变资本和可变资本也需要被转形为生产价格。此后数十年间，学术争论围绕此问题展开。

**马克思的文本证据（一手）：**
> 马克思自己在第三卷中承认了这个问题："在一个生产部门中的成本价格……可能偏离所使用的生产资料价值"，但他认为"这种偏离并不会改变成本价格是由商品中所包含的价值所决定这一事实"。

**辩护路线：**

| 派别 | 核心主张 |
|---|---|
| Paul Mattick | Böhm-Bawerk误解了马克思的方法——马克思从未声称单个商品按价值交换，而是认为价值决定价格和利润在**总量层面**运作的范围。总价值=总价格、总剩余价值=总利润在加总层次上成立 |
| "单一体系"解读（Freeman, Kliman） | 价值与价格在单一系统中同时决定，转形问题在正确理解马克思方法后消解 |
| 新李嘉图派（Sraffa, Steedman） | 承认转形有问题，但认为劳动价值论对于解释剥削是多余的——价格和利润率可直接从物质投入产出矩阵和实际工资中推出 |

> **可信度评估：** 此争议已持续120年以上，无共识。一手文本显示马克思意识到了问题但未给出完满解决方案；二手学术争论至今活跃。**状态：活跃争议，不试图调和。**

**来源：一手，马克思《资本论》第三卷；二手，Böhm-Bawerk (1896)、Bortkiewicz (1907)、Mattick (1983)、Kliman (2007)。**

### 4.2 利润率趋向下降的规律（第3篇）

> 马克思称这是"现代政治经济学的最重要的规律……从每一方面来说都是整个体系中最关键的规律"。

**规律本身：**
- 资本有机构成（c/v）在竞争中被迫提高
- 剩余价值只来自可变资本（v），而利润率 = s/(c+v)
- 因此，即使剩余价值率（s/v）不变甚至上升，利润率作为趋势下降
- 马克思数值示例：剩余价值率100%时，c从50增到400（v保持100），利润率从66.67%降到20%

**趋势而非数学宿命：**
马克思明确这是**趋势**（Tendenz），不是线性宿命。他列举了六个"起反作用的因素"：

| 反作用因素 | 机制 |
|---|---|
| 劳动剥削程度的提高 | 延长工作日、加速机器运转、提高剩余价值率 |
| 工资被压低到劳动力价值以下 | 事实上将部分劳动力价值转化为剩余价值 |
| 不变资本要素的便宜化 | 生产力提高使机器/原材料变便宜，减慢c的价值增长 |
| 相对过剩人口 | 大量失业使低有机构成、高剥削率的部门得以存在 |
| 对外贸易 | 廉价原材料和工资品输入；殖民地高利润率 |
| 股份资本的增加 | 股份公司中的资本只获得利息/股息，不参与利润率平均化 |

**矛盾（马克思自己承认）：**
> 利润率下降与利润量上升可同时发生：社会总资本扩大，绝对利润量增加，但相对投资回报率下降。马克思将此矛盾视为危机的来源之一——"过少资本"（无法获利投资的剩余资本）与"过多人口"（失业工人）并存。

**Engels编辑争议（二手，高可信度）：**
> 如前文所述，Heinrich指出恩格斯在编辑中强化了"崩溃"论的确定性。马克思的1864-65手稿中关于崩溃的表述是括号中的试探性说法，恩格斯删除了括号并移动了位置，使一个试探性的表述成为定论。

**来源：一手，《资本论》第三卷第三篇（恩格斯编） + MEGA II/4.2原始手稿；二手，Heinrich (1996)。**

### 4.3 生息资本、信用与地租（第4-6篇）

- **生息资本：** 资本作为商品出售——资本所有权与资本职能分离，产生利息。利息是剩余价值的一部分，在竞争中从利润中扣除
- **信用体系：** 银行、股份公司、虚拟资本。马克思称此为"资本主义生产方式在其自身范围内的扬弃"——私人资本向社会资本的过渡形式
- **级差地租与绝对地租：** 土地私有权使超额利润转化为地租，土地所有者从产业资本家手中分割剩余价值

## 五、《资本论》特有方法论

### 5.1 从抽象上升到具体

**一手来源——马克思在《大纲》导言（1857-58）中的阐述：**
> "具体之所以具体，因为它是许多规定的综合，因而是多样性的统一。因此它在思维中表现为综合的过程，表现为结果，而不是表现为起点……从抽象上升到具体的方法，只是思维用来掌握具体并把它当作一个精神上的具体再现出来的方式。"

关键区分：
- **研究方法：** 从具体的经验材料到抽象的规定（第一卷不直接展示这一步）
- **叙述方法：** 从抽象、简单的范畴上升到思维中的具体——"许多规定的丰富总体"（《资本论》的实际结构）

对应关系：商品 → 货币 → 资本 → 生产过程 → 流通过程 → 生产与流通的统一 → 在表面呈现的具体形式（利润、利息、地租）

**来源：一手，《政治经济学批判大纲》导言。**

### 5.2 逻辑与历史的统一

**一手来源——恩格斯在评论《政治经济学批判》时的表述：**
> "逻辑的研究方式是唯一适用的方式。但是，实际上这种方式无非是历史的研究方式，不过摆脱了历史的形式以及起扰乱作用的偶然性而已。"

**矛盾记录：** 马克思自己在《大纲》导言中表达了自相矛盾的立场：
- 一方面：抽象思维的道路"符合现实的历史过程"（如货币在历史上先于资本）
- 另一方面："把经济范畴按它们在历史上起决定作用的先后次序来排列是不行的……倒不如说，它们的次序是由它们在现代资产阶级社会中的相互关系决定的"

**当代争论（二手，活跃争议）：**
- **体系辩证法派**（Arthur, Reuten, Sekine）：《资本论》是纯共时性/结构性叙述，范畴代表同时存在的关系
- **逻辑-历史派**（Meek, 恩格斯）：逻辑序列反映了"被扬弃"形式中的历史发展
- **Vazjulin综合派：** 辩证叙述主要表现结构，历史在逻辑表现中被隐含地呈现

> **可信度评估：** 存在三种相互竞争的解读，无定论。马克思本人的文本提供对两种立场都能找到支持的证据。**状态：活跃争议。**

**来源：一手，《大纲》导言；二手，Ilyenkov, Arthur (2002), Rosdolsky (1977)。**

### 5.3 内在批判法

马克思的批判是**内在的**——不是从外部标准（道德、正义、人性）批判资本主义，而是从资本主义自身的范畴和矛盾中展开批判：

1. 从资本主义最基本的细胞形式（商品）出发
2. 揭示商品内在的矛盾：使用价值 vs. 交换价值，具体劳动 vs. 抽象劳动
3. 展示这些内部矛盾如何必然推动范畴运动——从商品到货币、货币到资本
4. 展示体系自身的逻辑展开如何产生不可解决的矛盾——这些矛盾表明资本主义的**历史限度**

> 这意味着马克思不是在"谴责"资本主义——他在展示资本的**历史暂时性**。正如他在《资本论》第一版序言中所说："我的观点是把经济的社会形态的发展理解为一种自然史的过程。"

**来源：一手，马克思原著 + 二手，Backhaus, Heinrich（德国"新马克思阅读"学派）。**

## 六、其他重要著作（作为《资本论》的背景和补充）

### 6.1 《1844年经济学哲学手稿》（1844，巴黎手稿）

**一手，未在马克思生前出版，1932年首次全文发表。**

**核心论点——"异化劳动"（Estranged Labor）的四重维度：**

1. **劳动者与劳动产品的异化：** "工人生产的越多，他能够消费的越少；他创造的价值越多，他自己越没有价值。" 劳动的产品作为异己的力量与工人对立。

2. **劳动者与劳动活动本身的异化：** 劳动不是自愿的而是被迫的。工人在劳动中"不是肯定自己，而是否定自己，不是感到幸福，而是感到不幸"。"动物的东西成为人的东西，而人的东西成为动物的东西"。

3. **人与类本质的异化：** 人类区别于动物的本质——自由、自觉的创造性活动——在异化劳动中被颠倒：生命活动仅仅成为维持肉体生存的**手段**。

4. **人与人的异化：** "人同自己的类本质相异化这一命题，说的是一个人同他人相异化。"——工人与资本家的对立来源于异化劳动。

**资本论视角的关联：**
- 1844手稿的"异化劳动"概念与后期《资本论》中的"商品拜物教"（物化）之间存在连续性——都描述人的产品/关系如何变成统治人的异己力量
- 但也有断裂：1844手稿依赖费尔巴哈式的人性假设（"类本质"/species-being），而后期《资本论》不再依赖这种哲学人类学基础，代之以从商品形式内在矛盾出发的内在批判
- **断裂论 vs. 连续论是马克思主义研究中的重大争论**（二手，活跃争议）

**来源：一手，《1844年经济学哲学手稿》；二手，Althusser（断裂论）, Lukács（连续论）。**

### 6.2 《关于费尔巴哈的提纲》（1845）

**一手，11条简短提纲，生前未发表，恩格斯1888年作为《费尔巴哈论》附录发表。**

核心命题：
- "从前的一切唯物主义——包括费尔巴哈的唯物主义——的主要缺点是：对事物、现实、感性，只是从客体的或者直观的形式去理解，而不是把它们当作人的感性活动，当作**实践**去理解。"
- 第11条："哲学家们只是用不同的方式**解释**世界，而问题在于**改变**世界。"

**来源：一手。**

### 6.3 《德意志意识形态》（1845-46，与恩格斯合著）

**一手，生前未全文发表，仅部分出版。**

**核心论点：**

**历史唯物主义系统阐述：**
1. **"不是意识决定生活，而是生活决定意识。"** ——历史唯物主义的基石
2. **"统治阶级的思想在每一时代都是占统治地位的思想。"** ——意识形态理论的核心
   - 支配物质生产资料的阶级同时支配精神生产资料
   - 意识形态把特殊阶级利益装扮为全社会的普遍利益
3. **分工是历史的引擎：**
   - "一个民族的生产力发展的水平，最明显地表现在该民族**分工的发展程度**上。"
   - 分工的每个阶段决定相应的所有制形式（部落所有制 → 古代公社所有制和国家所有制 → 封建的或等级的所有制）
   - 分工"把个人力量转化为物的力量"（物化/异化的起源）
   - **消灭分工** = 消灭私有制和阶级统治的条件
4. **国家是"虚幻的共同体"：** 国家声称代表普遍利益，但实际上"个人自由只是对那些在统治阶级范围内发展的个人来说是存在的"

**与《资本论》的关系：**
- 《德意志意识形态》提供了《资本论》的世界观基础——历史唯物主义的一般框架
- 但《资本论》的方法远为精细和辩证——不再停留于"物质基础决定上层建筑"的一般命题，而是展示资本主义生产方式特有的内在运动规律

**来源：一手，《德意志意识形态》。**

### 6.4 《共产党宣言》（1848，与恩格斯合著）

**一手。**

**核心论点（资本论视角的关联）：**
1. **"一切社会的历史都是阶级斗争的历史"** ——阶级斗争贯穿历史
2. **资产阶级的革命性与自我毁灭性：**
   - "资产阶级在它的不到一百年的阶级统治中所创造的生产力，比过去一切世代创造的全部生产力还要多，还要大。"
   - 但资本主义生产关系已变成生产力发展的"桎梏"——表现为"生产过剩的瘟疫"
   - "资产阶级首先生产的是它自身的掘墓人。"
3. **"一切坚固的东西都烟消云散了"（All that is solid melts into air）** ——资本主义的永恒变革动力
4. **劳动力的商品化（manifesto中的早期版本）：** 工人变成"像其他任何商品一样的商品"，"由于机器的推广和分工的加剧……工人成了机器的单纯的附属品"

**与《资本论》的对比：**
- 《宣言》是政治纲领——简洁、有力、面向行动
- 《资本论》是系统性的科学批判——在精细、辩证的结构中展开《宣言》中已蕴含的核心洞见
- 《宣言》中已出现剩余价值的萌芽理念：工人创造的"结晶化的社会劳动"被资本家占有

**来源：一手，《共产党宣言》。**

### 6.5 《路易·波拿巴的雾月十八日》（1852）

**一手。**

**核心论点：**
1. **经典开篇：** "黑格尔在某个地方说过，一切伟大的世界历史事变和人物，可以说都出现两次。他忘记补充一点：第一次是作为悲剧出现，第二次是作为笑剧出现。"

2. **阶级分析方法的具体运用（资本论视角关联）：**
   - 分析了1848-1851年法国各阶级的分裂与重组
   - **农民阶级："一袋马铃薯"**——客观经济条件使他们成为一个阶级（"数百万家庭在相同的经济条件下生活"），但主观上不是阶级（"没有形成任何一种政治组织"）。"他们不能代表自己，一定要别人来代表他们。他们的代表一定要同时是他们的主宰。"
   - **小资产阶级：** 他们的政治代表"不能超越小资产阶级在生活上没有超越的界限"
   - **流氓无产阶级：** 被波拿巴动员为"十二月十日会"——政治暴力的私人工具

3. **国家的相对自主性（对《资本论》方法的重要补充）：**
   - 波拿巴式的行政权力——"50万官僚和50万军队"——成为一个"缠绕法国社会全身并堵塞其一切毛孔的可怕的寄生机体"
   - 当没有单一阶级能直接统治时，国家获得相对的自主性，表现为凌驾于社会之上的独立力量
   - 这在理论上补充了《资本论》中较薄弱的国家理论部分

**来源：一手，《路易·波拿巴的雾月十八日》。**

### 6.6 《政治经济学批判大纲》（1857-58，Grundrisse）

**一手，生前未出版，1939-41年首次在莫斯科发表。**

> 此作品是《资本论》的**直接草稿**——两者方法论和内容高度重叠。

**《资本论》中未充分展开的关键内容：**

**"机器论片段"（Fragment on Machines，Penguin版pp.690-712）：**

1. **"一般智力"（General Intellect，马克思自创术语）：**
   > "固定资本的发展表明，一般社会知识已经在多么大的程度上变成了**直接的生产力**，从而社会生活的条件本身已经在多么大的程度上受到**一般智力**的控制并按照这种智力得到改造。"

2. **劳动时间不再是财富的尺度——资本主义的内在矛盾：**
   - 资本迫使科学技术发展 → 直接劳动在生产中作用降低
   - 但资本使**劳动时间**作为价值的尺度（交换价值的基础）
   - 因此："以交换价值为基础的生产崩溃了" → "资本本身是过程中的矛盾"
   - 马克思的预言性结论：当直接劳动不再是财富的主要来源后，"人不再被卷入生产过程，而成为生产过程的监督者和调节者"

3. **三阶段历史发展（比《资本论》更清晰）：**
   - 形式隶属 → 实际隶属（工场手工业）→ 实际隶属（机器大工业——一般智力）

4. **解放愿景：**
   - 一旦直接劳动不再是财富尺度，"**自由时间**"（disposable time）成为真正财富的标准
   - "**社会个人**"的全面发展——从事创造性、科学、艺术活动的全面发展的人——成为生产和财富的基石

> **当代意义（二手，高度活跃的阐释领域）：**
> 自主主义/后工人主义马克思主义者（Negri, Virno, Lazzarato）将"机器论片段"视为对当代"认知资本主义"的先知式描述——一般智力不再被固定在机器中，而是体现在语言、交流和情感的劳动中。
>
> **批评（二手）：** 有些学者认为"机器论片段"是马克思思想被抛弃的、"不成熟的"投机性产物——在《资本论》中他转向了更严格的价值理论和"总体工人"概念。**状态：活跃争议。**

**来源：一手，《政治经济学批判大纲》；二手，Virno (2004), Dyer-Witheford (1999)。**

### 6.7 《政治经济学批判》第一分册（1859）及其著名序言

**一手，马克思生前出版。**

**1859年序言中历史唯物主义的经典表述（资本论视角的核心背景）：**

> "人们在自己生活的社会生产中发生一定的、必然的、不以他们的意志为转移的关系，即同他们的物质生产力的一定发展阶段相适合的生产关系。这些生产关系的总和构成社会的经济结构，即有法律的和政治的上层建筑竖立其上并有一定的社会意识形式与之相适应的现实基础。物质生活的生产方式制约着整个社会生活、政治生活和精神生活的过程。**不是人们的意识决定人们的存在，相反，是人们的社会存在决定人们的意识。**"

> "社会的物质生产力发展到一定阶段，便同它们一直在其中运动的现存生产关系……发生矛盾。于是这些关系便由生产力的发展形式变成生产力的桎梏。那时社会革命的时代就到来了。"

**来源：一手，《〈政治经济学批判〉序言》。**

### 6.8 《法兰西内战》（1871）

**一手。**

**核心论点——"工人阶级不能简单地掌握现成的国家机器"：**

巴黎公社的经验使马克思得出了关于**国家形式**的最重要结论：

1. **打碎国家机器：** 不能像历次资产阶级革命那样从前者手中接过官僚-军事机器用于自己的目的——无产阶级必须**打碎**它

2. **公社作为"终于发现的"政治形式：**
   - 废除常备军，代之以武装的人民
   - 一切公职人员由选举产生、可随时罢免、领取工人工资
   - 不是议会式的（清谈馆），而是"**同时兼管立法和行政的工作机关**"
   - 政教分离
   - 公社不是不要国家统一——而是通过联邦式的公社体系组织国家统一

3. **"公社的真正秘密"：** 它"实质上是**工人阶级的政府**"，旨在"使劳动在经济上获得解放"——将生产资料从奴役工具转化为"自由联合的劳动工具"

> 恩格斯在1891年序言中明确鉴定："先生们，你们想知道无产阶级专政是什么样子吗？**请看巴黎公社。这就是无产阶级专政。**"

**与《资本论》的关系：**
- 《资本论》主要回答"资本主义的内在矛盾为何导致其崩溃"
- 《法兰西内战》回答了"取代它的应该是什么样的政治形式"
- 两者合起来构成马克思革命理论的完整结构

**来源：一手，《法兰西内战》。**

### 6.9 《哥达纲领批判》（1875）

**一手，生前未发表，1891年恩格斯发表。**

**核心论点（资本论视角的补充——过渡期理论）：**

1. **"无产阶级专政"的过渡时期：**
   > "在资本主义社会和共产主义社会之间，有一个从前者变为后者的革命转变时期。同这个时期相适应的也有一个政治上的过渡时期，这个时期的国家只能是无产阶级的革命专政。"

2. **共产主义社会的两个阶段——对《资本论》的超越：**

   **低级阶段（后来列宁称为"社会主义"）：**
   - 生产资料已社会所有，但"在经济、道德和精神方面都还带着它脱胎出来的那个旧社会的痕迹"
   - 分配原则："**按劳分配**"——劳动者从社会领回（扣除社会基金后）与劳动贡献等价的产品
   - **"这里通行的就是调节商品交换的同一原则"** ——等价交换。因此这仍然是**"资产阶级权利"**——平等的标准应用于不平等的个人，必然产生不平等的结果

   **高级阶段（"完全的共产主义"）：**
   > "在迫使个人奴隶般地服从分工的情形已经消失……在随着个人的全面发展，他们的生产力也增长起来，而集体财富的一切源泉都充分涌流之后，——只有在那个时候……社会才能在自己的旗帜上写上：**各尽所能，按需分配！**"

3. **对拉萨尔主义的批判：**
   - "劳动是一切财富的源泉"→ 自然同样是财富源泉，"一个社会主义的纲领不应当容许这种资产阶级的说法"
   - "铁的工资规律"→ "如果这个理论是正确的……我这二十年来的著作就不仅是白费，而且是无保留地默认了拉萨尔的理论"

**来源：一手，《哥达纲领批判》。**

## 七、马克思自创的核心术语和概念

| 术语 | 出处 | 定义 | 一手/二手 |
|---|---|---|---|
| **剩余价值**（Mehrwert） | 《资本论》第一卷 | 劳动力使用价值超过其自身价值的差额——工人无偿劳动的价值表现 | 一手 |
| **不变资本 / 可变资本**（konstantes/variables Kapital） | 《资本论》第一卷 | 不变资本=生产资料（不增殖价值）；可变资本=工资（劳动力）（增殖价值的唯一来源） | 一手 |
| **商品拜物教**（Warenfetischismus） | 《资本论》第一卷第一章 | 人与人之间的社会关系表现为物与物之间关系，且这种表现具有客观结构性基础 | 一手 |
| **生产方式**（Produktionsweise） | 多处 | 生产力与生产关系的统一体——特定历史阶段的经济结构形式 | 一手 |
| **生产力 / 生产关系**（Produktivkräfte/Produktionsverhältnisse） | 《德意志意识形态》、《〈政治经济学批判〉序言》 | 生产力 = 劳动资料+技术+劳动者技能；生产关系 = 所有制+财产关系+分工中的人与人的关系 | 一手 |
| **经济基础 / 上层建筑**（Basis/Überbau） | 《〈政治经济学批判〉序言》 | 基础 = 生产关系总和构成社会经济结构；上层建筑 = 法律政治制度和意识形态形式 | 一手 |
| **劳动力**（Arbeitskraft） | 《资本论》第一卷 | 区别于"劳动"——指人的劳动能力（作为商品出售），不是劳动本身 | 一手（马克思自认为是最重要的理论贡献之一） |
| **抽象劳动 / 具体劳动**（abstrakte/konkrete Arbeit） | 《资本论》第一卷第一章 | 具体劳动 = 特定质的有用劳动；抽象劳动 = 一般的、无差别的人类劳动力的消耗——商品价值的实体 | 一手 |
| **资本有机构成**（organische Zusammensetzung des Kapitals） | 《资本论》第一卷 | 由技术构成决定并反映技术构成变化的价值构成 c/v | 一手 |
| **一般智力**（General Intellect） | 《大纲》"机器论片段" | 物化在机器体系中的集体社会知识——直接生产力 | 一手 |
| **异化劳动**（entfremdete Arbeit） | 《1844年经济学哲学手稿》 | 劳动和劳动产品成为统治劳动者的异己力量 | 一手（但前期概念，后期较少使用） |
| **无产阶级专政**（Diktatur des Proletariats） | 《哥达纲领批判》《法兰西内战》 | 从资本主义到共产主义的过渡时期中劳动阶级的政治统治形式 | 一手 |
| **社会必要劳动时间**（gesellschaftlich notwendige Arbeitszeit） | 《资本论》第一卷 | 在现有社会正常生产条件下、社会平均劳动熟练程度和强度下生产某种使用价值所需的劳动时间 | 一手 |
| **实际隶属 / 形式隶属**（reelle/formelle Subsumtion） | 《资本论》第一卷、《大纲》 | 形式隶属 = 工人成为雇佣劳动者但劳动过程不变；实际隶属 = 劳动过程本身被资本主义改造（机器体系） | 一手 |

## 八、反复出现≥3次的核心信念（真信念判定）

> 判断标准：在马克思生前不同时期、不同著作中反复出现，不以单一段落为准。

### 1. 历史唯物主义 —— 信念确认
出现于：《德意志意识形态》（1845）→ 《政治经济学批判序言》（1859）→ 《资本论》第一卷序言（1867）→ 《哥达纲领批判》（1875）等。
> "人们的存在决定意识"贯穿始终，从未撤回或明显修正。**判定：核心真信念。**

### 2. 阶级斗争作为历史的动力 —— 信念确认
出现于：《共产党宣言》（1848）→ 《雾月十八日》（1852）→ 《法兰西内战》（1871）→ 致魏德迈信（1852，声明"不是我的发现"但确认其核心性）
> 从1848到1871，贯穿全部政治著作。**判定：核心真信念。**

### 3. 剩余价值理论与剥削 —— 信念确认
出现于：《雇佣劳动与资本》（1849，萌芽形式）→ 《大纲》（1857-58）→ 《资本论》全三卷（1867-1894）
> 从1840年代末到生命终结，是《资本论》的核心论题。**判定：核心真信念，毕生事业的中心。**

### 4. 资本主义必然崩溃并被共产主义取代 —— 信念确认
出现于：《共产党宣言》（1848）→ 《资本论》第一卷（"剥夺者被剥夺"）（1867）→ 《哥达纲领批判》（1875，详述过渡阶段）
> 从未撤回。但在《资本论》中，论证方式从《宣言》的历史必然性宣言转变为了对"历史趋势"（Tendenz）的分析。**判定：核心真信念，但随着成熟而细化。**

### 5. 商品拜物教与物化 —— 可能在后期的转换
出现于：《1844年手稿》（"异化"语言）→ 《德意志意识形态》（"物的力量"）→ 《资本论》第一卷（"商品拜物教"，1867）→ 《资本论》第三卷（"三位一体公式"，1894）
> 早期用费尔巴哈式语言（异化），后期发展出独立的、基于商品形式分析的概念（拜物教）。**判定：核心真信念，但概念工具经历了显著演化。**

### 6. 辩证法 —— 信念确认
出现于：《1844年手稿》（黑格尔批判）→ 《资本论》第一卷第二版跋（1873，明确论述"我的辩证方法"）
> 马克思在第二版跋中明确说："我的辩证方法，从根本上来说，不仅和黑格尔的辩证方法不同，而且和它截然相反。"但"我公开承认我是这位伟大思想家的学生。"**判定：核心真信念——方法论基石。**

### 7. 国家是阶级统治的工具 —— 信念确认
出现于：《德意志意识形态》（1845-46）→ 《共产党宣言》（1848）→ 《雾月十八日》（1852）→ 《法兰西内战》（1871）
> 贯穿始终，但后期对国家"相对自主性"的分析有所深化（《雾月十八日》）。**判定：核心真信念，但随经验分析而细化。**

## 九、思想谱系——影响、批判与超越

### 9.1 黑格尔（G.W.F. Hegel, 1770-1831）

| 继承 | 批判与超越 |
|---|---|
| **辩证法：** 历史通过矛盾运动向前发展。马克思称黑格尔辩证法中的"合理内核" | **唯心主义颠倒：** "在黑格尔那里，辩证法是倒立着的。必须把它倒过来。"——不是"绝对精神"的自我运动，而是物质生活的矛盾推动历史 |
| **历史主义：** 每个历史时代都有独特的内在原则 | **把概念当作独立主体的"实体化"：** 实际活着的感性的人才是历史的真正主体 |
| **异化：** 《精神现象学》中的Entfremdung概念被马克思重铸 | **异化从意识问题变为物质-经济问题** |

> **恩格斯的概括：** 黑格尔的体系是资产阶级的意识形态高峰——一种**"革命的思维方法被保守的体系所压倒"**的矛盾统一。
>
> **列宁的判断（《哲学笔记》）：** 不读懂黑格尔的《逻辑学》，就完全不能理解《资本论》。

**来源：一手，马克思《资本论》第二版跋 + 《1844年经济学哲学手稿》黑格尔批判部分；二手，列宁 (1914)、Ilyenkov (1960)。**

### 9.2 费尔巴哈（Ludwig Feuerbach, 1804-1872）

| 继承 | 批判与超越 |
|---|---|
| **唯物主义：** 人是自然的产物，上帝是人的本质的投射——"主谓颠倒"的方法 | **"直观的"而非实践性：** "从前的一切唯物主义……不是从实践去理解"（《关于费尔巴哈的提纲》第1条） |
| **宗教批判：** "人创造了宗教，而不是宗教创造了人" | **抽象的人的本质：** 费尔巴哈把"人"视为抽象的、非历史的本质。马克思认为人的本质是"社会关系的总和" |
| **"颠倒的方法"模型：** 马克思借用了费尔巴哈的"主谓颠倒"逻辑来颠倒黑格尔 | **没有看到宗教苦难的社会-经济根源：** "费尔巴哈没有看到，'宗教感情'本身是社会的产物" |

**来源：一手，《关于费尔巴哈的提纲》（1845） + 《德意志意识形态》第一部分。**

### 9.3 亚当·斯密（Adam Smith, 1723-1790）

| 继承 | 批判与超越 |
|---|---|
| **劳动价值论基础：** 商品的真实尺度是劳动 | **价值理论不一致：** 斯密在劳动价值论和"成本相加"论（工资+利润+地租）之间摇摆 |
| **剩余来源于生产而非交换：** 突破重农学派和重商主义 | **没有区分劳动和劳动力：** 马克思认为这是决定性突破——斯密未能识别剩余价值的独特来源 |
| **社会分工分析：** 分工是生产力的驱动力 | **非历史化：** 把资本主义范畴（工资、利润、地租）当作永恒的自然形式 |
| | **"物与物的关系"：** 斯密分析的是物之间的经济关系，掩盖了人与人之间的剥削关系 |

**来源：一手，《资本论》第一卷中对斯密的引用 + 《剩余价值理论》（《资本论》第四卷草稿）。**

### 9.4 大卫·李嘉图（David Ricardo, 1772-1823）

马克思对李嘉图的态度比他之前的任何经济学家都更加尊重——称李嘉图为"古典政治经济学的完成者"。

| 继承 | 批判与超越 |
|---|---|
| **严谨的劳动价值论：** 李嘉图比斯密更一致地坚持劳动时间决定价值 | **没有区分价值和交换价值：** 将价值直接等同于交换价值，忽略了使用价值这个维度 |
| **分配作为核心问题：** 社会总产品如何在地主、资本家、工人之间分配 | **剩余价值未被识别：** 分析了"利润"但从未将其源头识别为无酬剩余劳动 |
| **利润来源于生产：** 利润产生于生产过程本身 | **无危机理论：** 相信萨伊定律（供给创造自己的需求），看不到资本主义内在危机趋势 |
| **"运动规律"：** 李嘉图的价值理论是马克思发现资本"运动规律"的出发点 | **非历史化：** 和斯密一样，把资本主义当作永恒——从未问"为什么劳动表现为价值？为什么产品表现为商品？"（这恰恰是马克思拜物教分析和价值形式分析的起点） |

> 马克思在《资本论》第一卷中有一句著名评价：李嘉图"有意识地把阶级利益的对立、工资和利润的对立、利润和地租的对立当作研究的出发点，因为他天真地把这种对立看作社会的自然规律。但这使资产阶级的经济科学达到了它不可逾越的界限。"**来源：一手，《资本论》第一卷第二版跋。**

**来源：一手，《剩余价值理论》；二手，Dobb (1973)。**

### 9.5 法国社会主义的影响

马克思与恩格斯在《共产党宣言》中批判了"空想社会主义"（圣西门、傅立叶、欧文等），但承认他们的历史贡献——"这些著作抨击了现存社会的全部基础。因此，它们提供了启发工人意识的极为宝贵的材料。" 马克思自认为的突破：从**空想**到**科学**——不是从理想出发改造社会，而是从资本主义的内在矛盾中推导出它被超越的必然性。

## 十、重要矛盾与争议（不调和）

| 矛盾/争议 | 一手证据 | 评估 |
|---|---|---|
| **早期马克思 vs. 后期马克思——断裂还是连续？** | 1844手稿用"异化劳动"（基于费尔巴哈式类本质假设），《资本论》用"商品拜物教"和"剩余价值"（基于商品形式内在分析） | 活跃争议。Althusser（"认识论断裂"）vs. Lukács（连续论）。一手文本提供两种解读的证据 |
| **《资本论》中价值理论：实体论 vs. 形式论** | 马克思说抽象劳动是价值的"实体"，但也说价值"只有在交换中才能表现" | 活跃争议。"New Reading of Marx"学派（Heinrich, Backhaus）强调形式/交换的优先性，传统派强调生产/实体的优先性 |
| **利润率下降规律：规律还是趋势？** | 马克思自己说有时称为"规律"有时称为"趋势"，并承认多种反作用因素可能抵消甚至超过规律本身 | 持续争议。经验证据不确定——部分研究显示二战后利润率确实下降，但因果关系不明 |
| **转形问题的逻辑一致性** | 见上文第四节详尽讨论 | 120年+的争议，无共识 |
| **革命时机：西方无产阶级革命 vs. 资本主义适应性** | 《宣言》（1848）预判革命即将来临——170多年后的今天，西方资本主义仍有强大适应性 | 马克思本人后来也承认革命的长期性。恩格斯晚年（1895 "导言"）转向"合法斗争"策略——引起争议：是战术调整还是修正主义？ |
| **殖民主义与资本主义：既是进步的又是暴力的** | 马克思在论印度时既谴责英国殖民主义的"卑劣"，又称它是"历史的无意识的工具"——把印度从"停滞的"亚洲生产方式中拖出来 | 持续的殖民主义辩论——Marx的"进步主义"叙事遭受后殖民理论家严厉批判 |

## 十二、《资本论》核心分析框架总图

```
  商品（细胞形式）
    ├── 使用价值 ↔ 交换价值（内在矛盾）
    ├── 具体劳动 ↔ 抽象劳动（劳动二重性）
    ├── 价值形式分析：简单 → 扩大 → 一般 → 货币
    └── 商品拜物教：人与人的关系表现为物与物的关系
    
    ↓ 必然运动（内在矛盾推动）
    
  货币（价值的一般等价形式）
    ↓ 劳动力成为商品
    
  资本（自我增殖的价值）
    ↓
  
  ┌─────────────────────────────────────────┐
  │  生产过程（第一卷）                       │
  │  · 不变资本(c) + 可变资本(v) → c+v+s   │
  │  · s = 剩余价值（无酬劳动）              │
  │  · 绝对 + 相对剩余价值                   │
  │  · 资本积累 → 有机构成↑ → 相对过剩人口  │
  │  · 原始积累（暴力分离）                  │
  │  · 历史趋势："剥夺者被剥夺"              │
  └─────────────────────────────────────────┘
    ↓
  ┌─────────────────────────────────────────┐
  │  流通过程（第二卷）                       │
  │  · 三种循环形式（统一中的连续性）        │
  │  · 两大部类：I（生产资料）II（消费资料）│
  │  · 再生产图式（简单/扩大）               │
  │  · 流通中断 → 危机的形式可能性           │
  └─────────────────────────────────────────┘
    ↓
  ┌─────────────────────────────────────────┐
  │  总过程（第三卷）——接近"表面"           │
  │  · 剩余价值 → 利润                       │
  │  · 价值 → 生产价格（转形问题！）         │
  │  · 利润率向下趋势（Tendenz）             │
  │  │  └── 六种反作用因素                   │
  │  │  └── 矛盾：利润量↑ vs. 利润率↓       │
  │  · 生息资本 + 信用 → 虚拟资本            │
  │  · 地租（级差 + 绝对）                   │
  │  · "三位一体公式"批判：                  │
  │    资本-利润，土地-地租，劳动-工资       │
  │    → 资本主义的"神秘化完成"             │
  └─────────────────────────────────────────┘
    ↓
  结论：资本主义生产方式的历史暂时性
    ├── 内在矛盾 → 周期性危机
    ├── 生产力社会化 vs. 占有私有化 → 基本矛盾
```
