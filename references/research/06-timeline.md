# 卡尔·马克思完整时间线（以《资本论》为核心）

---

## 导读说明

本时间线以《资本论》的酝酿、写作、出版和影响为核心线索，将马克思生平、思想转折、重大历史事件和后世传播融为一条主线。所有关键思想转折点均以 **【转折点】** 标注。

**结构导航：**
- 第一章：早年与思想形成（1818--1843）
- 第二章：历史唯物主义的奠基（1844--1848）
- 第三章：伦敦流亡与经济学研究的深化（1849--1856）
- 第四章：《资本论》核心写作期（1857--1867）
- 第五章：晚年修订、政治实践与未竟事业（1868--1883）
- 第六章：恩格斯编辑遗稿（1883--1895）
- 第七章：《资本论》全球传播史（1872--2003）
- 第八章：21世纪的复兴（2008--2017）

---

## 第二章：历史唯物主义的奠基（1844--1848）

| 年份 | 事件 | 与《资本论》的关系 / 意义 | 标注 |
|:---:|------|---------------------------|:---:|
| 1844.02 | 恩格斯的《国民经济学批判大纲》发表于《德法年鉴》（马克思主编）。 | 被马克思称为"批判经济学范畴的天才大纲"，直接影响马克思转向政治经济学研究。马克思在《资本论》中多次引用此文。 | |
| 1844.04--08 | 撰写《1844年经济学哲学手稿》（巴黎手稿）。提出异化劳动理论：工人与其劳动产品、劳动活动、类本质及他人相异化。 | 《资本论》最早的思想萌芽。异化概念后来在《资本论》中发展为商品拜物教理论——人与人的社会关系披上了物与物的关系的外衣。该手稿迟至1932年才首次全文发表。 | **【转折点 2】** |
| 1844.08.28 | 在巴黎雷让斯咖啡馆与恩格斯第二次会面。十天深谈后两人发现"在所有理论领域完全一致"。 | 这是两人终生友谊与合作的开端，恩格斯后来成为《资本论》最重要的批评者、资助者和遗稿编辑者。 | **【转折点 3】** |
| 1845.02 | 马克思与恩格斯合著的第一部著作《神圣家族》出版。 | — | |
| 1845 春 | 撰写《关于费尔巴哈的提纲》。"哲学家们只是用不同的方式解释世界，问题在于改变世界。" | 被恩格斯称为"新世界观天才萌芽的第一个文件"。 | **【转折点 4】** |
| 1845.02 | 被法国政府驱逐，迁居布鲁塞尔。 | — | |
| 1845 夏 | 与恩格斯首次同赴英国曼彻斯特和伦敦，研究经济学，结识宪章派领袖。 | — | |
| 1845--1846 | 与恩格斯合著《德意志意识形态》（生前未发表，1932年首次全文出版）。 | 首次系统阐述历史唯物主义：生产力与生产关系、经济基础与上层建筑的辩证关系。这是《资本论》全部经济学分析的方法论地基。 | **【转折点 5】** |
| 1847.07 | 出版《哲学的贫困》（法文）。批判蒲鲁东。 | 马克思第一部公开发表的经济学著作。"经济范畴只不过是生产的社会关系的理论表现。" | |
| 1847.11--12 | 在伦敦出席共产主义者同盟第二次代表大会。受委托与恩格斯起草宣言。 | — | |
| 1848.02.24 | 《共产党宣言》在伦敦以德文出版。 | 宣告"至今一切社会的历史都是阶级斗争的历史"，预言资本主义的灭亡与无产阶级的胜利。《宣言》是马克思全部理论的"应用版"，其经济学论证将在《资本论》中完成。 | **【转折点 6】** |
| 1848.03 | 1848年革命爆发。马克思被比利时驱逐，先后回巴黎、回科隆。 | 革命中断了经济学研究。马克思从书斋走向街垒，获得后来深刻影响《资本论》分析的直接政治经验。 | |
| 1848.06.01 | 在科隆创办并主编《新莱茵报》。 | — | |
| 1849.05.16 | 被普鲁士政府驱逐。《新莱茵报》以红色油墨印行最后一期。 | — | |

## 第三章：伦敦流亡与经济学研究的深化（1849--1856）

| 年份 | 事件 | 与《资本论》的关系 / 意义 | 标注 |
|:---:|------|---------------------------|:---:|
| 1849 秋 | 流亡伦敦，此后在此度过余生。极度贫困中恢复经济学研究。 | 被迫告别直接革命的马克思转向"详细分析他想要推翻的经济体系"（Ralph Miliband）。这一转向产生了《资本论》。 | **【转折点 7】** |
| 1850--1853 | 在大英博物馆阅览室写下24册共1250多页的"伦敦笔记"。 | 系统研究国民经济史、货币流通、地租理论、工资理论。这是《资本论》的庞大资料库。 | |
| 1850 | 撰写《1848年至1850年的法兰西阶级斗争》。 | — | |
| 1851 | 在致恩格斯的信中乐观估计只需"五个星期"即可完成著作（实际耗费了16年以上）。 | — | |
| 1852 | 出版《路易·波拿巴的雾月十八日》。 | 详细分析了国家机器的阶级性质——1950年代后《资本论》关于国家的讨论因而不再重复这一论证。 | |
| 1851--1862 | 为《纽约论坛报》撰稿（当时世界上发行量最大的英文报纸），共发表约500篇文章。 | 新闻写作收入是马克思家庭的主要经济来源之一，使他得以坚持经济学研究。也为《资本论》积累了关于英国工厂立法、殖民地经济、亚洲社会等方面的实证素材。 | |
| 1855.01.16 | 小女儿爱琳娜出生。 | — | |
| 1855.04.06 | 年仅8岁的儿子埃德加尔因胃热病夭折，给马克思带来深重打击。 | — | |

## 第四章：《资本论》核心写作期（1857--1867）

| 年份 | 事件 | 与《资本论》的关系 / 意义 | 标注 |
|:---:|------|---------------------------|:---:|
| **1857.08--1858.05** | **撰写《1857--1858年经济学手稿》（Grundrisse，《大纲》）。**共8个笔记本，约50印张。 | **《资本论》第一稿。**1857年世界经济危机爆发，马克思"发狂似的通宵总结"经济学研究。在极端贫困和疾病中完成。制定了"六册计划"：资本、土地所有制、雇佣劳动、国家、国际贸易、世界市场。核心发现：(1)劳动的二重性（具体劳动与抽象劳动）；(2)劳动与劳动力的区分；(3)剩余价值理论（绝对剩余价值与相对剩余价值）。该手稿还包含关于自动化、机器和"一般智力"的惊人预见。手稿完全未知近一个世纪，直至1939--1941年才在莫斯科首次出版。 | **【转折点 8】** |
| **1859.06** | **出版《政治经济学批判》第一分册。**仅含"商品"和"货币"两章。 | 著名"序言"中精炼阐述历史唯物主义："物质生活的生产方式制约着整个社会生活、政治生活和精神生活的过程。不是人们的意识决定人们的存在，相反，是人们的社会存在决定人们的意识。"列宁认为此书与《资本论》第一卷同等重要。原计划分六册，但后续各分册均未出版——马克思改变了写作方案。 | **【转折点 9】** |
| **1861--1863** | **撰写《1861--1863年经济学手稿》。**包含23个笔记本，约200印张。 | **《资本论》第二稿。**其中包含著名的《剩余价值理论》——一部完整的剩余价值学说史（后由考茨基编辑为"第四卷"出版）。1862年马克思决定以"《资本论》"为总标题，以"政治经济学批判"为副标题出版。 | |
| **1863--1865** | **撰写1863--1865年经济学手稿。** | **《资本论》第三稿。**用两年半时间完成三卷《资本论》全部理论部分的详尽草稿（包括第二卷和第三卷的手稿）。这是恩格斯后来编辑第二、三卷所依据的主要材料。 | |
| 1863.11.30 | 母亲亨丽埃特·马克思在特里尔去世。 | — | |
| **1864.09.28** | **国际工人协会（第一国际）在伦敦圣马丁堂成立。**马克思出席并当选总委员会委员。 | 马克思起草了《成立宣言》和《临时章程》。他将大量的时间和精力投入第一国际的工作（通信、起草决议、调解内部矛盾），同时坚持《资本论》手稿的最终精炼。这是理论与实践并行的高峰期。 | |
| 1866.01--1867.03 | 对《资本论》第一卷进行最终定稿，将庞大的手稿精炼为"艺术性的整体"。 | "我为了为工人争得每日8小时的工作时间，我自己就得工作16小时。" | |
| 1867.04 | 马克思亲自携带书稿从伦敦赴汉堡，交付出版商奥托·迈斯纳。 | — | |
| **1867.08.16** | **凌晨两点，校对完最后一张印张，致信恩格斯："这一卷完成了。其所以能够如此，我只有感谢你！没有你为我作的牺牲，我是决不可能完成这三卷书的巨大工作的。"** | — | |
| **1867.09.14** | **《资本论》第一卷（德文第一版）在汉堡正式出版。**印数1000册。 | 副标题："政治经济学批判"。全书六章。马克思本人在致友人的信中说："这无疑是向资产者（包括土地所有者在内）脑袋发射的最厉害的炮弹。"马克思时年49岁。 | **【转折点 10】** |

## 第五章：晚年修订、政治实践与未竟事业（1868--1883）

| 年份 | 事件 | 与《资本论》的关系 / 意义 | 标注 |
|:---:|------|---------------------------|:---:|
| 1868--1871 | 深入修订《资本论》，大量搜集新材料（尤其是俄国土地关系和美国资本主义农业资料）。 | 马克思的研究范围远超《资本论》第一卷已涉领域。他阅读了大量俄文著作，研究了数学（微积分）、地质学、农艺学、人种学——这种永不满足的求知欲也是《资本论》未能完成的原因之一。 | |
| **1871.03--05** | **巴黎公社起义及其血腥镇压。** | 对《资本论》传播产生重大影响：(1)1871年《资本论》销售量激增至前几年的四倍（约200册），第一版几近售罄；(2)马克思因在报刊上被指认为"公社领袖"而声名大噪；(3)公社的失败促使法国流亡出版商拉沙特尔与马克思签约，使法文版《Le Capital》得以诞生；(4)马克思在《法兰西内战》中对公社经验的总结深化了《资本论》关于国家、商品拜物教和合作劳动的分析。 | |
| 1871.06 | 出版《法兰西内战》（国际工人协会总委员会宣言）。 | 提出核心论断："工人阶级不能简单地掌握现成的国家机器，并运用它来达到自己的目的。"这一思想后来被马克思在《资本论》法文版中进一步阐明。 | |
| **1872** | **《资本论》第一卷德文第二版（分册出版至1873年）。** | 重大修订：全书从六章扩展为七篇二十五章；首次独立设置"商品拜物教"一节；价值形式分析被彻底重写；对劳动价值论做了更精确的表述。这被视为马克思生前最重要的自我修订版。 | |
| **1872.03** | **《资本论》第一卷俄文版出版（洛帕廷与丹尼尔逊合译）。** | 《资本论》**全球第一个外文译本**。初版3000册。沙皇书报检查官因认定此书"太难，没有人会读"而放行——结果很快售罄，在俄国知识界引起巨大反响。 | |
| **1872--1875** | **《资本论》法文版（Le Capital）分期出版（共44个分册）。** | 马克思亲自逐页修订，远非简单的翻译。他在法文版中：(1)首次明确区分"资本积聚"与"资本集中"（这一区分后来被恩格斯纳入后期德文版）；(2)强调信用在积累中的作用；(3)将"工业较发达国家为较不发达国家展示了未来景象"的论断限定于"西欧各国"，留下非西方发展道路的可能性空间。马克思声称法文版"具有独立于原版的科学价值"。 | |
| 1872.09 | 第一国际海牙代表大会。巴枯宁被开除，总部迁往纽约。 | 马克思与巴枯宁无政府主义的决裂。 | |
| 1875 | 撰写《哥达纲领批判》（生前未发表，1891年由恩格斯公开发表）。 | 区分共产主义社会的低级阶段与高级阶段，提出"各尽所能，按需分配"。 | |
| 1878后 | **健康急剧恶化，被迫实质上放弃《资本论》后续各卷的写作。** | 肝病、慢性支气管炎、头痛、神经衰弱。马克思说"不能工作就等同于死刑判决"。他在最后几年仍大量阅读笔记（银行制度、货币市场、俄国和美国农村关系、地质学、人种学），但产出极少。 | |
| 1881.12.02 | 妻子燕妮·冯·威斯特法伦去世，马克思受到毁灭性打击。 | — | |
| 1883.01.11 | 长女燕妮·卡罗琳在巴黎近郊去世，年仅38岁。 | — | |
| **1883.03.14** | **卡尔·马克思在伦敦寓所安详去世，享年64岁。** | 书桌上摊着《资本论》最后一章的手稿——题为"阶级"——写到一页半便中断。恩格斯在葬礼致辞："正如达尔文发现有机界的发展规律一样，马克思发现了人类历史的发展规律……但这在他身上远不是主要的。马克思首先是一个革命家。"葬于伦敦海格特公墓。 | |

## 第六章：恩格斯编辑遗稿（1883--1895）

| 年份 | 事件 | 与《资本论》的关系 / 意义 | 标注 |
|:---:|------|---------------------------|:---:|
| 1883.03--05 | 恩格斯开始整理马克思遗稿。发现马克思留下了约1700页手稿，包含《资本论》第二卷的十份不同时期的草稿（最早可追溯至1865年）、第三卷一份1864--1865年的主要手稿，以及大量零散的笔记和摘录。 | — | |
| 1883 | 《资本论》第一卷德文第三版（恩格斯编辑）。 | 恩格斯将马克思在法文版中做的部分（但非全部）改动纳入德文第三版。 | |
| **1885.05** | **《资本论》第二卷"资本的流通过程"德文第一版出版（恩格斯编辑）。** | 恩格斯从十份手稿中选择、比对、拼接，编辑原则是"尽可能用马克思自己的话表达马克思的新成果"。距离第一卷首版已近18年。 | |
| 1887 | 《资本论》第一卷英文版出版（穆尔与艾威林翻译，恩格斯校订）。 | 首个完整的英文全译本。马克思小女儿爱琳娜校正引文。 | |
| 1890 | 《资本论》第一卷德文第四版（恩格斯编辑）。 | 目前流传最广的版本。恩格斯综合了德文第二版、法文版（部分改动）和德文第三版。 | |

| **1894.10** | **《资本论》第三卷"资本主义生产的总过程"德文第一版出版（恩格斯编辑）。** | 编辑极为困难。马克思1864--1865年的主手稿被恩格斯称为"极不完全"的初稿，"在若干地方只是思维在形成过程中写下的一些札记"。特别是第五部分（关于信用、利息和企业主收入），恩格斯三度尝试重写才勉强完成。出版时恩格斯已74岁，视力严重衰退。 | |
| **1895.08.05** | **恩格斯在伦敦去世，享年74岁。** | 他生前委托考茨基编辑《资本论》第四卷（即《剩余价值理论》）手稿，并指定伯恩斯坦为全部文稿的遗嘱执行人。恩格斯耗费了他人生的最后11年来完成这项工作——列宁说，没有恩格斯就没有《资本论》第二卷和第三卷。 | |

## 思想转折点总索引

| 编号 | 年份 | 事件 | 核心意义 |
|:---:|:---:|------|----------|
| 1 | 1843 | 撰写《黑格尔法哲学批判》 | 发现市民社会决定国家，非国家决定市民社会。转向政治经济学研究的逻辑起点。 |
| 2 | 1844 | 撰写巴黎手稿（1844年经济学哲学手稿） | 异化劳动理论——《资本论》商品拜物教理论的思想前身。 |
| 3 | 1844.08 | 在巴黎与恩格斯第二次会面 | 终生友谊与政治-理论合作的开始。恩格斯此后成为《资本论》最重要的对话者、资助者与遗稿编辑者。 |
| 4 | 1845 春 | 撰写《关于费尔巴哈的提纲》 | "问题在于改变世界。"实践概念的哲学奠基。 |
| 5 | 1845--1846 | 撰写《德意志意识形态》 | 历史唯物主义首次系统表述——《资本论》全部经济学分析的方法论地基。 |
| 6 | 1848.02 | 发表《共产党宣言》 | 资本主义必然灭亡的宣告——其经济学论证将在《资本论》中完成。 |
| 7 | 1849 秋 | 永居伦敦，开始系统经济学研究 | 从革命实践被迫转向理论分析。"剥夺了参加革命机会的革命者转向分析他想推翻的经济制度。" |
| 8 | 1857--1858 | 撰写《大纲》(Grundrisse) | 《资本论》第一稿。核心发现：劳动二重性、劳动力与劳动的区分、剩余价值。 |
| 9 | 1859 | 出版《政治经济学批判》第一分册 | 历史唯物主义的经典表述。"人们的社会存在决定人们的意识。" |
| 10 | 1867.09.14 | 《资本论》第一卷正式出版 | 马克思主义政治经济学体系的完成形态的开端。"向资产者脑袋发射的最厉害的炮弹。" |
